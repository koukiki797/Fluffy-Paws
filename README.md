# 🌿 Fluffy Paws | Exotic Shorthair Breeder

エキゾチックショートヘア専門ブリーダー **Fluffy Paws** の公式サイトです。
「愛情と自然の中で育てる」をコンセプトに、子猫の紹介やブリーダーの想いを伝えるデザインを目指しました。

## 🌐 公開URL
https://koukiki797.github.io/Fluffy-Paws/

## ✨ 特徴
- **ナチュラルなデザイン**: セージグリーンとリネンカラーを採用し、猫の穏やかな性格を表現。
- **レスポンシブ対応**: スマートフォン、タブレット、PCのすべてのデバイスで最適に表示されます。
- **ギャラリー機能**: 清潔な飼育環境や子猫の日常を視覚的に伝えます。
- **お問い合わせフォーム**: ユーザーがスムーズに相談できるシンプルな UI。

## 🛠 使用技術
- **HTML5**: セマンティックなマークアップによる構造化。
- **CSS3**: Flexbox や Grid を用いたモダンなレイアウト。
- **JavaScript**: ハンバーガーメニューの開閉やフォームのバリデーション制御。
- **Google Fonts**: 『Shippori Mincho』『Cormorant Garamond』を使用。

## 📂 フォルダ構成
- `index.html`: メインページ
- `exotic_breeder.css`: スタイルシート
- `main.js`: インタラクション制御
- `image/`: サイト内で使用する画像リソース

本プロジェクトは学習・デモ用です。画像等の無断転載はご遠慮ください。
